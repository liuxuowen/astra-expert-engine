# 留学知识库总览

本目录用于沉淀卡米拉在群内持续输出的留学知识点。

原则：
- 不为每个新知识点单独建文件
- 先归纳，再分类
- 控制子文档数量，保持结构稳定
- 知识库文档与记忆文档允许重复，不做去重

当前结构：

1. `knowledge-base/consulting-framework.md`
   - 咨询推进逻辑
   - 学生与家长高频问题
   - 顾问分析框架
   - 选校、定位、活动判断的通用方法

2. `knowledge-base/application-materials-and-timeline.md`
   - 申请材料清单
   - 美本申请基本进度
   - 时间轴与阶段任务
   - 顾问推进中的交付物

3. `knowledge-base/requirements-and-policy-signals.md`
   - 语言与标化趋势
   - 顶尖院校分数参考线
   - QuestBridge 国际生政策
   - 敏感专业额外审核与风险提醒

适用范围：
- 以美本咨询为核心
- 对通用申请逻辑优先抽象
- 专业差异先不做过细拆分，避免碎片化
