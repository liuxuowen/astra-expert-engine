# 本科申请知识库

按国家/地区分类的本科申请院校速查与顾问话术沉淀。

## 目录结构

```
undergraduate/
├── usa/          # 美国本科
├── uk/            # 英国本科
├── sg/            # 新加坡本科（已有：NUS 2026/2027）
├── hk/            # 中国香港本科
├── au/            # 澳大利亚本科
├── ca/            # 加拿大本科
├── eu/            # 欧洲（德国、荷兰、法国等）
├── other/         # 其他国家/地区
└── nus-undergraduate-2026-2027.md  # NUS 示例文件（待迁移）
```

## 使用说明
- 每个国家/地区对应一个子目录
- 院校文件以「院校名-申请季」命名，如 `nus-undergraduate-2026-2027.md`
- 顾问话术、速查版、申请要求等内容可直接写入对应院校文件
- 后续可按国家/地区分别汇总成该国的综合申请指南

## 命名规范
- 目录名：ISO 3166-1 alpha-2 小写（us → usa，gb → uk）
- 文件名：`{院校简称}-{类型}-{申请季}.md`
  - 例：`nus-undergraduate-2026-2027.md`
  - 例：`ucl-graduate-2026.md`
  - 例：`hkust-undergraduate-2026-2027.md`
